# Frontend Assessment — Responsive UI

A responsive frontend application built using HTML, CSS, and JavaScript (ES6+) based on the provided Figma design.

---

## Features

* Responsive Hero Slider
* Swiper.js Integration
* Pagination Dots
* Navigation Arrows
* Autoplay Slider
* Infinite Loop
* Product Listing Grid
* Responsive Product Cards
* Mobile Responsive Design
* Dynamic Product Rendering using JavaScript
* Clean UI Layout

---

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* Swiper.js

---

## Responsive Design

### Desktop

* 4 Product Cards Per Row
* Card Size: 462 × 710

### Mobile

* 2 Product Cards Per Row
* Card Size: 173.5 × 276
